Chat - express, socket.io, axios
-------------

#### client 
```
npm install -g express
npm install -g express-generator
express "이름" --view=ejs
npm install
```

#### client run
 > npm start

<br>

#### server 
```
npm install 
npm install socket.io axios --save
```
#### server run 
> node server.js
